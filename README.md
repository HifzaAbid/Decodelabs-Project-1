# GlamGlow - Premium Makeup Store

An elegant, semantic static website developed for a luxury cosmetics brand. This project serves as a practical milestone completed during my Frontend Development Internship at DecodeLabs.

## Features

- Sleek Header: A single-row design with the brand logo on the left and navigation menu items on the right.
- Hero Banner: Includes a sophisticated cursive headline with a coordinating call-to-action button.
- Product Grid: A uniform 3x3 layout displaying exactly three products per row, complete with descriptions and ratings.
- Custom Palette: Utilizes curated pink and purple hues suited for a premium beauty brand identity.

## Technical Architecture

- HTML5: Built with structural semantic tags (header, nav, main, article, footer) for clean code.
- CSS3: Engineered using CSS Grid for macro card layout and Flexbox for micro-alignments.
